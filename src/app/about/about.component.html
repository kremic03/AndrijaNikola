About CineMax

CineMax is a web prototype for viewing movies and booking tickets online.

This application was developed as part of the Client Web Applications course at Singidunum University.

Libraries used in the development:

  • Angular 19
  • Angular Material
  • Axios HTTP Client
  • SweetAlert2 for notifications

Features

  • Browse movies currently in theaters
  • Search for movies by title, genre, or date
  • View detailed movie information
  • Book tickets with multiple seating options
  • Manage your reservations
  • Rate movies you've watched
